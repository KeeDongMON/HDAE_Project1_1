키보드 입력처럼 수정

swL : 좌측 방향 지시등
swR : 우측 방향 지시등
swP : 자율 주차 시작 버튼

블루투스 송신
(x 명령);(y 명령);(왼쪽깜빡이);(오른쪽깜빡이);(주차모드)
ex) w;x;0;0;0

(x 명령);(y 명령)

직진'w';'x'
후진 's';'x'
우회전 'w';'d'
좌회전 'w';'a'
우후진's';'d'
좌후진's';'a'

(왼쪽깜빡이);(오른쪽깜빡이);(주차모드)
